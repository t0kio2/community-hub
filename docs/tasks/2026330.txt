原因

- HTMLが描画されず text/plain 1バイトになっています。Deviseコントローラの親が API 用（ActionController::API）で、テンプレート描画スタックがないためです。

実施した対応

- WebController を追加して HTML 用スタックを用意。
    - 追加: api/app/controllers/web_controller.rb
- Devise の親コントローラを WebController に変更。
    - curl -i http://localhost:3001/admin/auth/sign_in
    - もしくはサイズ/タイプのみ確認:
    - `curl -s -o /dev/null -w 'code:%{http_code} type:%{content_type} bytes:%{size_download}\n' http://localhost:3001/admin/auth/sign_in`
- ブラウザキャッシュの影響を避けたい場合（必須ではありません）:
    - curl -i "http://localhost:3001/admin/auth/sign_in?ts=$(date +%s)"

これで Content-Type: text/html でログインフォームHTMLが返るはずです。まだ text/plain のままなら教えてください。
